# Netflix Movies & TV Shows Dataset - Exploratory Data Analysis

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to identify trends, patterns, and insights related to Netflix content.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Project Tasks

1. Data Loading and Inspection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights and Recommendations

## EDA Questions Answered

* Movies vs TV Shows distribution
* Top countries producing Netflix content
* Content added per year
* Most common content ratings
* Most popular genres
* Top directors on Netflix
* Top actors on Netflix

## Visualizations

The project includes:

* Pie Chart
* Bar Chart
* Line Chart
* Histogram
* Scatter Plot
* Heatmap

## Key Insights

* Movies dominate Netflix's content library.
* The United States contributes the highest number of titles.
* Netflix experienced significant growth after 2015.
* TV-MA is the most common content rating.
* International and drama content are highly represented.

## Conclusion

This project successfully analyzed Netflix content using Python-based data analysis tools. Through data cleaning, visualization, and exploratory analysis, valuable insights were obtained regarding Netflix's content distribution, growth trends, and global reach.
